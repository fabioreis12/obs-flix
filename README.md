# 🎬 Obs-Flix: Backend de Alta Performance com Observabilidade Nativa

Este projeto simula um backend de streaming (estilo Netflix) desenvolvido em **Python (FastAPI)**, focado no monitoramento e métricas de produção usando **Prometheus** e **OpenMetrics**.

## 🚀 Como Rodar o Projeto
O projeto está totalmente containerizado. Para subir o ambiente completo (API + Monitoramento), basta um comando:

```bash
docker-compose up --build
